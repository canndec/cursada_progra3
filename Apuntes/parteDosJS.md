## JavaScript II / Control de flujo, estructuras de control, condicionales y bucles I

control de flujo: como, en donde y bajo que condiciones se ejecutan instrucciones de un programa

# condicionales
- condicionales: / if / else if / else
- operadores ternarios: / condicion ? si : no /


# bucles I
- for: (incializacion; condicion; incremento){ cod a ejecutarse por iteracion}
- while: (condicion) {cod a ejecutarse por condicion verdadera}
- do...while: do{cod a ejecutar} while {condicion}

# control de flujo avanzado
- break: salir inmediatamente de un bucle
- continue: salta a lasiguiente iteracion, omitiendo el cod restante para esa iteracion

- switch: ejecutar un bloque de cod correspondiente al caso coincidente 
    * (expresion) case val1: cod a ejecutar, break ... ;
